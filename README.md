# Simon 学习助手

基于西蒙学习法的专注学习计时器，帮助你高效学习。

## ✨ 功能特性

- 🍅 **番茄钟计时** — 25分钟专注 + 5分钟休息 + 30分钟长休息
- 📝 **费曼笔记** — 每3个番茄钟后强制记录学习笔记，支持编辑和删除
- 📅 **学习计划** — 三层计划结构（总计划→周计划→日任务）
- 📊 **学习统计** — 连续打卡、每周专注时间、完成率
- 🎨 **学习活跃度热力图** — GitHub 风格，支持年份切换
- 🌐 **多语言支持** — 中文/英文
- 💻 **系统托盘** — 最小化到托盘，快捷键操作

## 📦 安装方式

### 下载安装包（推荐）

1. 前往 [Releases](https://github.com/MgGaWin/simon-releases/releases) 页面
2. 下载最新的 `Simon学习助手_v1.1.0_Setup.exe`
3. 双击运行安装程序
4. 按照向导完成安装

### 系统要求

- Windows 10/11
- 无需安装 Python 或其他依赖

## 🎯 使用方法

1. **开始学习** — 点击"开始"按钮，进入25分钟专注时间
2. **休息** — 专注结束后自动进入5分钟休息
3. **长休息** — 每3个番茄钟后进入30分钟长休息
4. **费曼笔记** — 长休息前记录学习笔记（可跳过）
5. **查看统计** — 切换到"统计"标签查看学习数据和活跃度热力图

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Space` | 开始/暂停计时 |
| `Ctrl+Shift+N` | 跳过当前阶段 |
| `Ctrl+Shift+S` | 停止计时 |
| `Ctrl+1` ~ `Ctrl+5` | 切换标签页 |
| `Ctrl+W` | 隐藏到托盘 |
| `Ctrl+Q` | 退出程序 |

## 📁 数据存储

所有数据存储在本地 JSON 文件中：

- `data/plan.json` — 学习计划
- `data/daily_log.json` — 每日记录
- `data/notes.json` — 费曼笔记
- `data/settings.json` — 用户设置
