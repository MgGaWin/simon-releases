# Simon 学习助手

基于西蒙学习法的专注学习计时器，帮助你高效管理每日学习计划。

> "你只需要坐下来，点开始，其他的交给工具。"

## ✨ 功能特性

- **🍅 番茄钟计时** — 25分钟专注 + 5分钟休息 + 30分钟长休息
- **📝 费曼笔记** — 每3个番茄钟后强制记录学习笔记，巩固知识
- **📋 学习计划** — 三层计划结构（总计划→周计划→日任务）
- **📊 学习统计** — 连续打卡天数、每周专注时间、任务完成率
- **📅 日历热力图** — 绿/黄/红可视化每日学习进度
- **🌐 多语言支持** — 中文/英文界面
- **💻 系统托盘** — 最小化到托盘，不干扰工作

## 📦 安装方式

### 下载安装包（推荐）

1. 前往 [Releases](https://github.com/MgGaWin/simon-releases/releases) 页面
2. 下载最新的 `Simon学习助手_v1.0.0_Setup.exe`
3. 双击运行安装程序
4. 按照向导完成安装（可选择创建桌面快捷方式）

### 系统要求

- Windows 10/11
- 无需安装 Python 或其他依赖

## 🎯 使用方法

### 基本操作

1. **开始学习** — 点击"开始"按钮，进入25分钟专注时间
2. **休息** — 专注结束后自动进入5分钟休息
3. **长休息** — 每3个番茄钟后进入30分钟长休息
4. **费曼笔记** — 长休息前记录学习笔记（可跳过）
5. **查看统计** — 切换到"统计"标签查看学习数据

### 三层界面

| 层级 | 触发方式 | 内容 |
|------|----------|------|
| 第一层 | 始终可见 | 系统托盘图标，显示当前状态 |
| 第二层 | 左键点击托盘 | 悬浮小窗口，快速查看和操作 |
| 第三层 | 双击托盘/右键菜单 | 完整主面板（5个标签页） |

### 番茄钟模式

| 阶段 | 时长 | 说明 |
|------|------|------|
| 🍅 专注 | 25分钟 | 高质量专注时间 |
| 💤 短休息 | 5分钟 | 离开屏幕放松 |
| ✍️ 费曼总结 | 手动 | 每3个番茄钟后触发 |
| ☕ 大休息 | 30分钟 | 上下半场之间 |

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Space` | 开始/暂停计时 |
| `Ctrl+Shift+N` | 跳过当前阶段 |
| `Ctrl+Shift+S` | 停止计时 |
| `Ctrl+1` ~ `Ctrl+5` | 切换标签页 |
| `Ctrl+W` | 隐藏到托盘 |
| `Ctrl+Q` | 退出程序 |

## 📸 界面预览

- **今日页面** — 圆盘倒计时 + 任务列表
- **日历页面** — 热力图可视化学习进度
- **统计页面** — 连续天数、每周时长、完成率
- **计划页面** — 管理学习任务
- **设置页面** — 自定义时间参数

## 📁 数据存储

所有数据存储在本地 JSON 文件中：

- `data/plan.json` — 学习计划
- `data/daily_log.json` — 每日记录
- `data/notes.json` — 费曼笔记
- `data/settings.json` — 用户设置
